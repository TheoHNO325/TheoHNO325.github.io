4.3节

知识点

回顾内容：的解的情况与的关系

线性组合与线性表示

n维向量可以看作是n维空间中的一个向量，一些n维向量的线性组合可以形成一个新的空间

定理4.2.1 （线性方程组与线性相关的关系）：向量可由向量组a1-an线性表示 线性方程组有解；表示法有唯一或者无穷多两种情况，对应着方程组的解(k1-kn)的情况。

所以求线性表示方法（求k）的问题可以转化为求解线性方程组的问题，详见课本例4.2.3。解的情况可由增广矩阵的秩是否被b增大了（A出现全0的时候b不是0）

定义:等价向量组 设有两个向量组

若（I）中每个向量都可由（II）线性表示，则称（I）可由（II）线性表示（废话）.若两者可以相互线性表示（注意！1可被2表示不一定能证明2可被1表示），则等价。等价具有自反性、对称性、传递性。

题目：判断是否等价。根据上面求线性表示的方法，若要判断等价，则每个I中的向量都能被II表示，反之亦然，让两者轮流做A和B（由一些b构成）判断有没有解即可，详见课本例4.2.4.

线性相关与线性无关

定义 一些向量a1-an，若存在k1-kn使加权和为0，则线性相关。反之则线性无关。

定理4.2.2 A=(a1,a2,...as)，即A的列向量线性相关

Ax=0有非零解,det(A)=0 （对应了线性表示的k）

A的秩小于s

虽然不是定理但很重要 n+1个n维向量一定线性相关

几何直观：三个三维向量如果共面，则它们可以看作三维空间上的一个张成了二维平面的向量，其中总有一个废物能被另外两个加出来，此时为线性相关；如果不共面，则三人总是分道扬镳，各有立意（可以想想为什么？）

并且，如果有四个三维向量（推论4.2.2）则多出来的那一个总会走其他三个的老路，因为三个线性无关的三维向量已经足够涵盖整个三维空间

题目：判断是否线性相关。详见例4.2.8。

该题思路一：判断Bx=0是否有非零解 -> 把a待入b -> 合并a的同类项得到系数矩阵P -> P没有非0解

思路二：先把B=AP表示出来 -> Bx=APx=0 都没有非0解

思路三：P可逆，故B,A秩相等->B的秩=列数（定理422）->线性无关

定理4.2.3 延续4.2.2的定义：向量a1-as线性相关

至少存在一个ai可被剩下的s-1个线性表示

定理4.2.4十分重要！ 向量a1-as线性无关，加上b就相关，则b可由a1-an线性表示且表示法唯一。（唯一的证明：设有两个表示法，相减=0）

定理4.2.5 部分组差-->整体组差；整体组好-->所有部分组都；有个0分的-->整体必定差

向量组的秩

回顾矩阵的秩的求法：化为简化阶梯形看首非零元个数

定义:极大无关组 向量组U中的部分组可以“代表”全部且线性无关。注意: 1.它不唯一 2.U与自己的极大无关组等价 3.利用这点可以判断U中的向量是否线性相关 4. 若r(U)=r，则U中任意r个线性无关向量可作为U的极大无关组

求解方法：列向量变成矩阵，化为阶梯性，首非零元在的那些列可构成极大无关组，进一步化简可以得到剩下向量的线性表示方法。详见例4.3.1。

定义:向量组的秩 极大无关组的个数。（若U仅含零向量则为0）

定理4.3.1 r(A)=A的列秩=A的行秩（秩总是从最小的那个方阵看的）

虽然不是定理但是看起来也很重要 矩阵初等行变换之后，虽然前后列向量不等价（相当于左乘P），但是列向量之间的线性关系不改变（Ax=0的解不改变）。

向量组的秩与矩阵的秩的关系

定理4.3.2 设有两个向量组

若（I）可由（II）线性表示，则 (1)s > r,(I)线性相关 (2)若(I)线性无关，s <= r

若（I）（II）等价，则s=r。

定理4.3.3 若（I）可由（II）线性表示，则r(I) <= r(II),I的极大无关组只能来自于II,（极大无关组与U等价），把极大无关组套到定理4.3.2里比较，得出结论I的极大无关组的数量肯定<=II的。

其实如果两者等价，则r(I)=r(II),一切又回到了原点。

定理4.3.4 矩阵的秩的变化（直接见书）

题型库

送分：求秩/已知秩求值/求极大无关组并表示/（习题4.3 1~3）

仔细：向量组矩阵的简化阶梯性（例4.3.1 & 习题4.3 9）

证明：

  1. 证明两向量组秩相同
    • 证明等价（反过来不可以！）
    • 能互相线性表示--r的比较
  2. 证明等价
    • 若为方阵：即证相互能线性表示--任选一个b放到a线性相关--也许定理4.2.4.（习题4.3A6）
  3. 证明线性相关/无关
    • 算Ax=0是否有非零解
    • 算秩（极大无关组向量数）是否=列向量数量（通常需要和另外一个向量组使用定理4.3.3比较）
    • 结论+1：若I能表示II：n维基本单位向量组，则I线性无关（习题4.3 A7& B2）

4.4节

知识点

定义：齐次巴拉巴拉 通解可以加可以乘（非齐次不行）

定义：基础解系 Ax=0的一组解向量（线性无关且可表示任何一个解）（解集的极大无关组）（所有不正交的通解构成解空间）

虽然不是定理但是习题A1,5 与基础解系等价的线性无关向量也是基础解系（证明思路：线性无关+每个都能做解+线性组合也能表示所有解）

定理4.4.1非常重要！ 如果系数矩阵A(m x n)的秩r(A)< n未知数的个数,则Ax=0肯定有非零解（也有基础解系）并且成员的个数是n-r个

证明：想象把[A|0]化成阶梯性，则最底下有n-r行全是0，后n-r个未知数遂可以随便取，加权构成前r个，这些权重向量即为基础解系，因为它们把后n-r个未知量分别连接到了前r个。（看书）

结构解的表示:

注意：基础解系不唯一，但是数量一定相同

定理4.4.2 基础解系可以由任意n-r个线性无关的解向量构成（联系解向量的线性组合仍然是解，故只要新的解向量组线性无关，怎么变都还是呢能做基础解系，见例4.4.3）

定义：非齐次巴拉巴拉 Ax = b. 解集三种情况：无解/唯一/无穷多. 性质1：两个特解的差为对应齐次的解 性质2 对应齐次的解+特解=特解

定理4.4.3 在Ax=b有解的前提下，任意一个解可表示为一个特解+对应齐次的一个解，也可表示为一个特解+基础解系结构式。Ax=0的解的数量决定了Ax=b，期中成绩决定了期末成绩（如果能活到期末的话）

空间

定义：数域 (书P142) 包含0和1,且对加减乘除封闭的数的集合,用F表示

例:R,Q,C是数域,Z,R+,无理数集不是数域

定义：线性空间 1.对定义的加法,乘法封闭.2.满足八条性质,和数域F的关系

需要满足的八条性质(证明时需一一验证): - 交换律 - 加法结合律 - 存在0元,0+a=a - 每个a有负元 - 存在1元,1a=a - 乘法结合律 - 乘法分配律1 - 乘法分配律2

例:F^n是F上的线性空间(自己封闭),这里的F可以是任何F.

一些课本上的表示:

  • :元素属于数域F的mxn矩阵的全体
  • C[a,b]定义在闭区间上一元连续函数的全体
  • 系数取自数域F的n次多项式全体(but似乎对x没有明确的要求)

性质: 看起来不是很重要,到时候再说

定义：子空间 V内的W也满足定义,对子空间

定义：span V内取定的向量的所有线性组合可以形成一个W(需要证明一下封闭)

定义：基 能张成V的极大无关组

定义：维数 dim(V)=r(V)=极大无关组中列向量的个数=V的维数

例:标准基,(1,0,0,0)之流十分标准

注意选择题:零空间dim=0没有基;无限维空间;有限维空间基不唯一,等等等,但维数是线性空间的

"可将V看成是由基向量生成的线性空间，而基向量就是线性 无关的生成元."

变换

现在对于一个V,我们假设已经得到它的两个基了.

定义：过渡矩阵与坐标变换公式 基1到基2要右乘A,则同一个x在基1下的坐标到基2要左乘个A-1

定义：坐标映射 选定V的基,用这个基当然能线性表示V中任意一点, 线性表示的具体化就是一串x1-xn,即为V中点在基下的坐标

该映射是个双射,由此引申出不同V之间的映射关系:同构

定义：同构 把V1中的α映射到V2中的β的双射(一一映射,可加可乘,因为都是线性空间),记作T

T(α) = β,α在V1中具有什么属性(like相关,基),β就有什么属性

同构具有自反,对称,传递性,维数为n的V与F^n同构

V1,V2维数相同 同构
