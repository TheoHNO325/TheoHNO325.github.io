7 指针和引用

引用

在定义引用时,必须要用某个现有的原始变量初始化,(int &person = student)并且不能再作为其他变量的引用.对引用操作等同于对原始变量的操作.输出引用的时候,得到的是地址.

两种出现"&"的情况:

  1. &的前面有类型符(int float):必然是对引用的声明,定义了一个引用.
  2. 没有:取变量的地址,左边可以用指针接住.然后可以用*指针在地址上改掉原来的变量的值(在函数中尤其需要)

不可以引用:int &...

  1. 数组 a[9]
  2. 指针 p (但是可以int& p=q;q为指针)
  3. 引用的引用 int & &p
  4. 前面的类型是void
  5. 用类型或者null初始化

经典的Swap:

Swap(int &a,int &b){ int t = a; // 引用本身是个值,带上&取地址 a = b;//修改引用,则原值也会变化,因为引用和原始变量共用一块地址 b = t; }

指针

&(引用):不是在声明的时候,也可被认为是取地址运算符.&i:变量i的地址

int pi = & i; 指针变量pi,初始化为i的地址. 两者互为逆运算,&对变量取地址,得到指向该变量的指针; 对地址取值,得到地址所指的变量.

三种安全的赋值方法:

  1. int *p = NULL;(需要引入头文件stddef.h)NULL=0的常量但可以直接赋值给指针,表示指针不指向任何内存区域

  2. int *p = new int; ... delete p;

  3. int *p = a;(a为初始化过的变量地址)

否则,若*p未赋值就改变值,将会随机改变一个内存地址的变量,十分危险

指针的运算: 加减:以其指向类型的字长为单位 相减:指针之间的距离 数组变量

相等:== != 值是否相等

8 数组

一维数组

定义了一块连续的空间，数组元素按序存放在这块空间中,数组名字是数组首元素的内存地址.空间的大小等于元素数*每个元素所占 的空间大小，如定义int a[10]，将占用了40个字节空间，因为每个整型数占 (1 Byte= 8 bit)4个字节。

初始化:初始化的长度短于要被初始化的数组元素数目，那么剩余元素被初始化为0.初始化的可以不定义长度.

动态编程:new & delete

共用体

使几个不同的变量共享同一段内存的结构.

共用体变量中起作用的成员是最后一次被赋值的成员，在对共用体变量中的一个成员赋值后，原有变量存储单元中的值就取代.

初始化共用体变量时,初始化表中只能有一个常量.各成员的地址都是同一地址.

不能对共用体变量名赋值，也不能企图引用变量名来得到一个值.

构造函数与析构函数

构造函数 是类的成员函数，函数名必须与类名相同.在创建对象时，使用给定的值来将对象初始化.可以带参数、可以重载，同时没有返回值.必须放在公有的成员函数里,也可放到保护里(如果定义的类仅用于派生其他类)

有了构造函数,我们就不再需要在创建对象之后再调用一遍函数了.(类似python的__init__(self))

可以copy的构造函数 甚至还可以将另一个对象的引用作为构造函数的参数,这样就直接复制了.

其实,如果不定义这个卡卡西函数,系统会自动生成一个隐含的.于是可以自动赋值.(A a3=a1;A a2(a1) //可以这样直接赋值)

如果没有定义构造函数,系统将自动产生一个缺省的构造函数A(){},此时我们不能在实例化时直接赋值. 不存在没有构造函数的对象,所有对象在定义时调用构造函数.

new运算符 动态建立对象

总之它就是这么定义的,不管什么构造在堆上还是栈上...new完要delete

析构函数 ~A(){} 撤消对象时(例:作用域退出/程序结束)由系统自动调用,不允许重载,不能带有任何参数，不能有返回值，不指定函数类型.

如果忘记定义析构函数了,系统会自动产生缺省的. 但如果用new了,必须显示定义一个来delete.

对象

类的变量(实例),具有实际的存储空间.

定义格式: 1. 类名 对象名1,对象名2,...;(局部对象) 2. 在定义完类的最后一个大括号之后 对象名1,对象名2;(全局对象)

调用对象的成员 "."

注意:用成员选择运算符“.”只能访问对象的公有成员，而不能访问对象的私有成员或保护成员。若要访问对象的私有的数据成员，只能通过对象的公有成员函数来获取。

时间差

  1. 对于全局定义的对象（在函数外定义的对象），在程序开始执行时，调用构造函数；到程序结束时，调用析构函数
  2. 对于局部定义的对象（在函数内定义的对象），当程序执行到定义对象的地方时，调用构造函数；在退出对象的作用域时，调用析构函数.
  3. 用static定义的局部对象，在首次到达对象的定义时调用构造函数;到程序结束时，调用析构函数;并且static只能定义一次!(所以才有能力只在程序结束时释放)
  4. 对于用new运算符动态生成的对象，在产生对象时调用构造函数，只有使用delete运算符来释放对象时，才调用析构函数。若不使用delete来撤消动态生成的对象，程序结束时，对象仍存在，并占用相应的存储空间，即系统不能自动地调用析构函数来撤消动态生成的对象.(new对象必须用delete来激法析构函数)

new 数组也是同理,且delete必须加上[],否则只能释放第0个元素.(delete []pa1;)

杂门

作用域 指程序中所说明的标识符在哪一个区间内有效，即在哪一个区间内可以使用或引用该标识符

五类: 块作用域、文件作用域、函数原型作 用域、函数作用域和类的作用域

块作用域 在块内说明的标识 符，只能在该块内引用.作用域在该块内from标识符的说明处to块的结尾处。

局部变量 在一个函数内部定义的变量(包括形参)或在一个块中定义的变量. 其作用域成为块作用域. 1. 主函数main中定义的变量，也只在主函数中有效，同样属于局部变量. 2. 不同的函数可以使用相同名字的局部变量，它们在内存中分属不同的存储区间，互不干扰.
3. 变量名相同,局部更优先

文件作用域 全局变量的作用域,在整个文件可以访问.缺省的作用范围:从定义全局变量的位置开始到该源程序文件结束.

全局变量 在函数外定义的变量.其作用域称为文件作用域,在整个文件中可以访问. 与块作用域的变量同名时,局部变量优先.

选做7:动态内存分配

new : 动态地在堆内存上分配内存，并返回指向该内存块的指针

int* ptr = new int; int* ptr = new int(5); //分配一个整数的内存并初始化为5 int* arr = new int[10]; int* c = new char[10]; int* arr = new int[10]{1,2,3,4}; // 初始化 int* ptr = nullptr // 传入空指针,安全,但也需要delete

delete:释放由 new 分配的内存

delete ptr; delete[] arr; //释放数组内存要用delete[]

如果忘记释放动态分配的内存，程序就会发生内存泄漏，导致内存浪费。

printf 格式化输出

来自gpt:

%[flags][width][.perc][F|N|h|l]type

在 C++ 中，printf 是一种用于格式化输出的函数。它使用一系列格式化标志来控制输出的外观。你提到的格式字符串 %[flags][width][.precision][length]type 就是 printf 输出格式的一个基本结构，每个部分的含义如下：

  1. flags（标志位）：
    • - : 左对齐（默认是右对齐）。
    • + : 强制显示符号（对于整数类型，如果是负数则显示负号，正数则显示加号）。
    • 0 : 用零填充输出（如果数字的长度不足指定宽度，则前面填充零）。
    • # : 对某些类型加上特殊符号，比如对浮点数显示e、对八进制数显示0前缀、对十六进制数显示0x。
    • space : 在数字前面显示一个空格（正数前加空格，负数则直接显示负号）。
  2. width（宽度）：
    • 这是输出字段的最小宽度。如果实际输出的内容宽度小于这个宽度，则根据标志位（比如 0 或 -）填充空格或零。如果实际内容宽度大于这个值，则会显示完整内容。
  3. .precision（精度）：
    • 对于浮点数，表示小数点后保留的位数。
    • 对于字符串，表示最大字符数。
    • 对于整数，表示输出最少的数字位数，使用零填充不足的部分。
  4. length（长度修饰符）：
    • h : 指定短整数类型（如 short 或 unsigned short）。
    • l : 指定长整数类型（如 long 或 unsigned long）。
    • ll : 指定长长整数类型（如 long long 或 unsigned long long）。
    • j : 指定适用于 intmax_t 类型的整数类型。
    • z : 指定适用于 size_t 类型的整数类型。
    • t : 指定适用于 ptrdiff_t 类型的整数类型。
  5. type（类型说明符）：
    • d 或 i : 整数类型（有符号）。
    • u : 无符号整数。
    • o : 八进制整数。
    • x 或 X : 十六进制整数（小写或大写字母）。
    • f : 浮动点数（浮点数，格式为固定小数点）。
    • e 或 E : 浮动点数（科学计数法）。
    • g 或 G : 浮动点数（自动选择 f 或 e，以最短方式显示）。
    • c : 单个字符。
    • s : 字符串。
    • p : 指针地址。
    • % : 输出一个百分号字符。

其他格式化输出：

printf 确实还支持一些其他的格式化输出方法，除了标准的整数、浮点数、字符串输出，常见的还有：

  • 宽字符和宽字符串：wchar_t 类型的字符可以使用 %lc 来输出，宽字符串可以用 %ls 来输出。
  • 指针：输出指针的地址，使用 %p 格式。
  • 格式指定：在特定场景下可以通过 * 来动态指定宽度和精度。
